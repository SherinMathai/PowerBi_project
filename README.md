# PowerBi_project
📊 Project Overview

This project is an interactive IPL (Indian Premier League) Analysis Dashboard built using Microsoft Excel. It analyzes historical IPL data (2008–2018) to uncover meaningful insights about teams, players, venues, and match outcomes.

The dashboard transforms raw cricket data into visually appealing and easy-to-understand insights using Excel’s powerful data analysis features.

🚀 Features
🏆 Team & Match Insights

Total matches played and wins by each team

IPL winners across seasons

Toss decision vs match outcome analysis

🌟 Player Performance

Top players with most Man of the Match awards

Performance trends of key players

📍 Venue Analysis

Top stadiums hosting the most matches

Venue-based match trends

📈 Interactive Dashboard

Dynamic charts and graphs

KPI cards for quick insights

Slicers/filters for interactive exploration

🛠 Tools & Technologies Used

Microsoft Excel

Pivot Tables

Pivot Charts

Slicers

Conditional Formatting

Data Cleaning & Transformation

Dashboard Design Principles

📂 Dataset

IPL match data from 2008 to 2018

Includes match results, teams, venues, toss decisions, and player awards
